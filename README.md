# Pokedex-paginador
Pokedex con Api y paginador

Link: <a href="https://nicolascarrica.github.io/Pokedex-paginador/">Pokedex-Demo</a>

## Para poder correrlo..

```
npm install
```

Una vez descargadas las dependencias se debe levantar el servidor

```
npn run serve
```
